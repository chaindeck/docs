## Getting started

With our UI, you can easily query chaindeck's data and retrieve results using simple sql query. To get started, follow these steps

1. Login to https://app.chaindeck.io/login 
2. Click on Authorize button towards the top right corner.
3. Fill details in Authorize modal.
4. Once authorized you can make query using chaindeck SQL UI.
