## Steps

To use `orca-pod-template`, you have to follow these steps respectively.

#### 1. Clone `orca-pod-template`

<CodeGroup>

```bash Terminal
git clone https://github.com/chaindeck/orca-pod-template.git
```

</CodeGroup>

#### 2. Start `orca-pod-template`

<CodeGroup>

```bash Terminal
node index.js
```

</CodeGroup>
